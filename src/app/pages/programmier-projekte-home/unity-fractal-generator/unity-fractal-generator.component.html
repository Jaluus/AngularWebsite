How does it work?

Im using a special technique called "Ray Marching" to render the fractals. Each fractal is "build" with Signed Distance Functions (SDF) , or so called "Distance Estimators", to get the current distance to the fractal. I use the SDFs of simple shapes to "build" more complex shapes such as the Menger Sponge. This is a dramatic oversimplificaton of the whole process, if you wish to learn more about the rendering technique of these fractals I can recommend videos of these Youtube channels
