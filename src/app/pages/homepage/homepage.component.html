A Sandbox Website

The primary idea of this website is to just be a Sandbox for my ideas, I have written some code in my freetime in Python and C++, but sadly its hard to show people.

A Website is the perfect way to showcase some of my stuff but to also learn some new stuff!

So I decided to learn Angular as the Frontend Framework and later implement my code on a Flask backend server for Python code - Mainly for the Neural Networks - aswell as a NodeJs server for JavaScript related Stuff.

At the moment I have written some simple algorithms like the Game of life, a Fractal Tree Generator and the A* Pathinding algorithm, give it a try, but please try not to break it, it might break your current tab in the browser if you try hard enough!

You can also take a look at my Repo with the link in the top right hand corner, the code for Angular is probably abysmal, but not as bad as the algorithms, they are extremly inefficent, but they work, so hey, don't fix what's not Broken.
