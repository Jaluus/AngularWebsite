Tensorflow Object Detection
How to Download and Install
  • Download the APK here
  • Make sure you enabled "install from unknown sources" in the Android settings.
  • Give the App the right to access your camera.
  • You are ready to detect some Objects!
Additional Information

This app is Android only, because Apple is quite strict on external apps and im not gonna put this app onto the Appstore, sorry!

For the App I used some code from this GitHub Repo, for example the layout code.
